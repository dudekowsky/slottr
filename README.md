# Slottr

Simple React + Rails for slot booking management.

Things you may want to cover:

## Requirements
* Ruby 3.1.2
* Node 16.18.1

## Get started
Go in the base folder and run:
```
bundle
npm install
rails db:create
rails db:migrate
rails db:seed
rails s
```
then in another terminal tab run
```
npm run dev
```

## Specs
Just run `rspec`
